# AwasDitabrak 🚗💨

**AwasDitabrak** (Watch Out for the Crash!) is a simple yet addictive 2D arcade game built using Python and the Pygame library. Test your reflexes as you navigate through traffic, collect power-ups, and try to beat your highest score!

## 🌟 Features
- **Smooth Lane Transitions:** Fluid movement between lanes for a better gaming experience.
- **Dynamic Difficulty:** The game gets faster and more challenging as your score increases.
- **Power-up System:** Collect blue orbs to gain a **Shield** that protects you from one collision.
- **Persistent High Score:** Your best score is saved locally in a `.txt` file so you can always track your progress.
- **Asset Integration:** Uses custom images for cars and the road environment.

## 🎮 Controls
| Key | Action |
|-----|--------|
| **A** | Move Left |
| **D** | Move Right |
| **R** | Restart Game (on Game Over) |
| **Q** | Quit Game (on Game Over) |

## 🛠️ Prerequisites
Before running the game, ensure you have Python installed and the `pygame` library:

```bash
pip install pygame
```
